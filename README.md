# No YouTube Shorts (v1.2.0)

A powerful, privacy-respecting Chrome extension that removes YouTube Shorts from your feed and redirects Shorts links to the regular video player.

- **Cleans Feeds**: Removes Shorts shelves, carousel items, and search results.
- **Hides Navigation**: Removes the "Shorts" tab and sidebar buttons.

### âœ¨ New in v1.2.0
- **Channel Whitelist**: Allow specific creators (e.g., `@MKBHD`) to show Shorts in your feed.
- **Context Menu**: Right-click any Short link to "Open as Regular Video".
- **Instant Redirect Mode (Beta)**: Optional setting to use advanced browser network rules for faster, instant redirection (note: disables redirect whitelist).

## Installation

1. Clone this repo
2. Open `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder

## Privacy

This extension does not collect any data. All settings are stored locally on your device.
